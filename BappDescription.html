A Burp Suite extension for out-of-band (OOB) testing called Interactsh Collaborator - Revised. It is an enhanced tool that effortlessly integrates with Interactsh, providing unique domains for OOB tests. It supports multiple protocols and features real-time monitoring. It not only seamlessly integrates with Burps default Collaborator, but also ensures the interface is user-friendly and easy to operate.

Features

Core Functionality

  • Interactsh Integration: Generate unique Interactsh domains for OOB testing
  • Multiple Protocol Support: DNS, HTTP/HTTPS, SMTP, FTP, LDAP, SMB
  • Real-time Monitoring: Automatic polling for interactions with configurable intervals
  • Burp Integration: Works alongside Burps native Collaborator

User Interface

  • Protocol Formatting: Clean, readable display for each protocol type with different presentation styles for DNS, SMTP, FTP, LDAP, SMB/Responder, and HTTP/HTTPS.
  • Interactive Table: Sortable, filterable interaction log
  • Unread Tracking: Visual indication of new interactions
  • Copy to Clipboard: One-click URL copying
  • Protocol Filtering: Filter by protocol type (All, HTTP, DNS, SMTP, LDAP, SMB, FTP)

Usage

Basic Workflow

  1. Go to the Interactsh tab in Burp Suite
  2. Use the Copy URL to clipboard function to copy the Interactsh URL
  3. Integrate the URL into your tests (e.g., inject into parameters, headers, etc.)
  4. Watch the interactions in real-time in the table

Buttons and Controls

  • Copy URL to clipboard→ Copies the current Interactsh URL to clipboard
  • Regenerate Interactsh Session→ Generates a new session with a new URL
  • Refresh→ Manually triggers a poll for new interactions
  • Clear log→ Removes all interactions from the table
  • Poll Time→ Shows the current polling interval (seconds)

Protocol Filtering

Use the filter buttons to only view specific protocol types:

  • All - Shows all interactions
  • HTTP - Only displays HTTP/HTTPS
  • DNS - Only displays DNS queries
  • SMTP - Only displays email interactions
  • LDAP - Only displays LDAP queries
  • SMB - Only displays SMB/Responder
  • FTP - Only displays FTP connections

Table Features

  • Click a row to view more information
  • Sort by column by clicking column headers
  • New interactions are highlighted in bold
  • Select cells/rows to copy data to clipboard
